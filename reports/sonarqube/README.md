# SonarQube Reports

This directory contains SonarQube analysis reports generated by the CI/CD pipeline.

## Report Types

| Report | Description |
|--------|-------------|
| `bugs-report.json` | Code bugs detected |
| `vulnerabilities-report.json` | Security vulnerabilities |
| `code-smells-report.json` | Code quality issues |
| `duplications-report.json` | Duplicated code blocks |
| `quality-gate-status.json` | Quality gate pass/fail status |

## Quality Gate Configuration

The quality gate fails the pipeline if:
- New Bugs > 0
- New Vulnerabilities > 0  
- New Security Hotspots Reviewed < 100%
- New Code Coverage < 80%
- New Duplicated Lines > 3%

## Viewing Reports

Reports are generated automatically during CI/CD pipeline execution. Access the full dashboard at your SonarQube server URL (e.g., `http://localhost:9000`).

## Scanned Files

- YAML configuration files (`*.yaml`, `*.yml`)
- Shell scripts (`*.sh`)
- Application source code
- Deployment manifests
